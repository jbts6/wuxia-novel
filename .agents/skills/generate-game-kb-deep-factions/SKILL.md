# generate-game-kb-deep-factions

Use this skill only against a published v5 base and the `factions` scope.
Submit a `factions-deep` deferred task bound to the base manifest hash.
Operations are limited to existing grounded factions: keep, merge, drop, or a
constrained patch. No invented faction, evidence quote, source reference, or
cross-category entity is allowed.

Produce overlay drafts only; never mutate the base five YAML files or accepted
evidence directly. Applying an overlay must reject a stale base and use an
atomic revision receipt.
