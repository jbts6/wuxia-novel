# 常量定义

## ID 规则

格式：全小写拼音，字间下划线。前缀：`char_` `faction_` `loc_` `skill_` `item_` `tech_`

## source_refs（必填）

每个实体必须携带：
```json
"source_refs": [{"chapter": 1, "line_start": 42, "line_end": 45, "text": "原文片段"}]
```
