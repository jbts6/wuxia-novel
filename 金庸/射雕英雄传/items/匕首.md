# 匕首

## 描述
黄蓉所携匕首，用以威胁吕文德、割断穆念慈绳索、划开帐篷；黄蓉的匕首，用以穿通密室气孔通风，危急时递给郭靖准备御敌

## 效果
- **attack**: 郭靖瞥见匕首想起杨铁心遗命，感叹尊长安排未必定须遵行
- **构陷工具**: 杨康以此匕首为证，将郭靖与全真七子联系起来，坐实'帮凶'之名
- **attack**: 瑛姑十馀年苦练当胸一刺，直没至柄，郭靖指力强劲才仅入肉半寸
- **tool**: 割断刺槐勾刺开辟道路，割断船缆升帆离岛
- **weapon**: 李萍以此割断郭靖手上绳索，随即转剑尖刺入自己胸膛
- **symbol**: 丘道长取名寓意不可忘靖康之耻
- **weapon**: 锋利短刃多用途，胁迫敌人割绳索均可
- **weapon**: 杨铁心怀中防身匕首，被丘处机空手夺去
- **tool**: 丘处机用此匕首在两柄短剑上分别刻下'郭靖''杨康'二字
- **attack**: 刺入陈玄风肚脐练门直没至柄，立时毙命
- **special**: 阳光下闪烁如彩虹，引起朱聪注意
- **attack**: 穆念慈含羞自刎以死抗争
- **harm**: 穆易挡格不及，匕首刺入其手掌致手背鲜血淋漓
- **attack**: 寒光闪闪的锋利匕首，可刺入人体
- **plot**: 柄刻杨康二字，是确认身份和杀夫仇人的关键信物
- **buff**: 柄上刻'郭靖'二字，为郭杨两家指腹为婚的信物
- **attack**: 拔出寒气扑面，穆念慈称赞好剑
- **sentimental**: 杨铁心与郭啸天为儿女订亲之物，包惜弱临终取出，具有婚姻信物意义
- **special**: 匕首外包裹着刻有九阴真经的陈玄风胸口人皮
- **工具**: 用以割臂放血救周伯通蛇毒，也曾刻字竹筒传信'一起活一起死'
- **threat**: 黄蓉以匕首抵胸威胁父亲，被欧阳锋蛇杖暗器击落
- **attack**: 匕首锋锐无比，可轻而易举刺穿鲨鱼头颅，刺中巨鲨咽喉后借上升之势划开腹部
- **utility**: 割两块五寸见方船帆写求救信号缚于白雕脚上
- **utility**: 割断小艇系索令小艇脱离大船逃离
- **attack**: 刺入郭靖左腰数寸深，拔出时鲜血如泉水喷涌
- **utility**: 用匕首穿通密室被尘土闭塞的气孔，使秽气散出
- **weapon**: 黄蓉将匕首递给郭靖，准备在有人旋开橱门时御敌
