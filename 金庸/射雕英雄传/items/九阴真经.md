# 九阴真经

## 描述
天下武学秘要，分上下卷，载有各种精妙武功和梵文怪文

## 效果
- **buff**: 载有九阴白骨爪、摧心掌等武功，练成后横行江湖无敌
- **debuff**: 无内功心法根基，练出的功夫偏于阴毒，有走火入魔之险
- **knowledge**: 记载九阴白骨爪、摧心掌等外门神功，但缺上半部内功根基
- **plot**: 刺于皮肉的经文是梅超风修炼的唯一依据
- **buff**: 天下武学至高秘笈，五气朝元、和合四象、三花聚顶等内功法门均载于此
- **knowledge**: 王重阳华山论剑赢得，临终传给周伯通，被黄药师抢走半部，又被黑风双煞盗去部分，欧阳克欲从梅超风处夺取
- **buff**: 包含普天下各家各派功夫的武学体系，奇幻奥秘神妙之极，但争夺此经者多丧命，练错则走入邪道
- **武学宝典**: 天下学武之人视为至宝，上下卷融会贯通后武功不可思议
- **修炼指导**: 上卷为道家修炼内功大道及拳经剑理，非克敌制胜之功；下卷为实用法门
- **lore**: 周伯通因教郭靖背诵而不自觉练成经中武功
- **narrative**: 周伯通双手夹住经书以内力压成碎片，千千万万片碎纸如蝴蝶飞散
- **debuff**: 照假经练功虚耗时日、劳而无功，反而身子受害
- **deception**: 郭靖书法拙劣反显真实，欧阳锋见其不知写法请教欧阳克，信以为真
- **special**: 手心向天改脚底向天，脚踏实地改手撑实地，气凝丹田改气凝胸口
- **knowledge**: 郭靖内功根柢的重要来源，欧阳锋认为留他终是祸胎
- **buff**: 易筋锻骨篇使郭靖黄蓉七日功力大增，洪七公据此悟恢复之法
- **special**: 经文末尾有拗舌赘牙的怪文，欧阳锋百思不得其解，洪七公用来拖延时间
