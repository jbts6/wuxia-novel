# 小红马

## 描述
汗血宝马，脚力天下无双，四蹄如飞犹似凌空御风，红色汗珠溅在雪地如樱花绽放；郭靖的坐骑，跟主人跳上桃花岛后在花树丛中走失。；郭靖坐骑脚力快捷，黄蓉骑此马北上犒劳蒙古大军又急驰而回

## 效果
- **buff**: 众人骑此马追赶杨康，双雕飞行虽速小红马竟追随得上
- **transport**: 从沪溪到铁掌山六十余里片刻即至
- **transport**: 电驰而出四足犹似凌空，极快到达目的地
- **坐骑**: 奔行迅速，两人同骑共驰日行极远，从江南西路直抵两浙南路东海之滨
- **mount**: 在山谷隘口助郭靖抵挡金兵，如飞般迎敌
- **mount**: 郭靖骑乘小红马在荒野中走来走去，苦苦思索破城杀敌到底该是不该。
- **transport**: 全速奔驰可甩脱欧阳锋轻功追击，穿越沼泽不陷
- **bond**: 自寻觅食避过战祸，主动找到主人舔脸亲热
- **mount**: 奔跑速度远超常马，韩宝驹的追风黄远远不及，只认郭靖一主
- **buff**: 脚力快极，两个时辰大跑不见费力，如箭离弦
- **buff**: 知主人之意，右膝轻碰即回头奔上土山
- **speed**: 长途疾驰后肩渗红汗而精神健旺，奔速如箭
- **rarity**: 西域大宛天马后裔，家马与野马交配而生
- **social**: 白驼山八女欲夺此马献给少主，彭连虎等群雄亦欲染指
- **speed**: 日行千里的宝马，奔跑极快任谁也追赶不上
- **mount**: 脚力快捷日行千里，驮二人毫不费力
