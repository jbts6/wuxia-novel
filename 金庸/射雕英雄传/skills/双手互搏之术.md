# 双手互搏之术

## 描述
周伯通所创之术，郭靖用以左右手分使不同招式，危急中配合一阳指指形吓退欧阳锋。

## 招式列表
| 招式 | 类型 | 描述 |
|------|------|------|
| [[一心二用]] | buff | 左手解索右手迎敌，丝毫不见局促，左右手可同时执行不同动作 |
| [[分手自救]] | special | 急危中以左右互搏之术分手相救，郭靖以此抵御瑛姑阴毒后着 |
| [[双桨互搏]] | special | 双手分使神龙摆尾推铁桨，左一桨右一桨，将铁舟推得如顺水而行 |
| [[左右分使]] | special | 周伯通所创之术，可左右手分使不同招式。郭靖在危急中左手挡胸护体，右手食指伸出模拟一阳指点向欧阳锋太阳穴，双管齐下。 |
| [[左右分击]] | attack | 一人化二，左右手各使不同招式同时攻击，郭靖以此术同时施展鸿渐于陆与亢龙有悔 |
| [[双手分打节拍]] | special | 郭靖脱去左脚鞋子在空竹上敲打节拍扰乱箫声，双手分打如同两人合力 |
