# 摔跤术

## 描述
郭靖的蒙古摔跤技艺，以骆驼扳、断山绞等招式近身缠住欧阳锋使其无法施展上乘武功

| 招式 | 类型 | 描述 |

| [[骆驼扳]] | attack | 左手从敌人左腋下穿出，在其后颈猛力扳落，手法巧妙，以骆驼之庞然大物亦不免颈骨断折为喻 |
