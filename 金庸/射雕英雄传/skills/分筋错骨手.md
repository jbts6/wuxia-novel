# 分筋错骨手

## 描述
洪七公制服欧阳克三名白衣手下时所用，打耳光同时卸脱其下颏关节。

## 招式列表
| 招式 | 类型 | 描述 |
|------|------|------|
| [[壮士断腕]] | attack | 拿敌手腕肘部，前送后扭，使右腕关节脱出，为分筋错骨手中拿手招式 |
| [[笑语解颐]] | attack | 捏向敌腮向外急拉，使下颚关节应手而脱，朱聪取其滑稽之名 |
| [[分筋错骨手]] | debuff | 扣住敌手腕脉门使其动弹不得，配合单刀削下迫敌弃枪 |
| [[脉门擒拿]] | debuff | 单骑冲阵时以分筋错骨手扣住都史左腕脉门将其擒下 |
| [[绞拿法]] | attack | 左手向上向右、右手向下向左，双手交叉而落，同时拿住对方双腕脉门 |
| [[扣拿穴法]] | attack | 伸食中两指扣拿对方腕后养老穴，反手钩擒抓向指关节 |
| [[拿筋错节]] | attack | 双手飞舞拿筋错节，招招不离对手全身关节穴道 |
| [[钩拿抓打]] | attack | 钩、拿、抓、打四式连环，招招凶猛狠辣的擒拿手法 |
