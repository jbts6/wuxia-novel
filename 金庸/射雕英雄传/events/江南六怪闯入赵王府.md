# 江南六怪闯入赵王府

## 描述
江南六怪追踪欧阳克至赵王府，翻墙而入。韩宝驹与南希仁首先攻击欧阳克，发现郭靖被梅超风挟持后大惊失色。六怪此前曾在塞北大漠围攻欧阳克，柯镇恶吃他一杖，朱聪以分筋错骨手扭断其左手小指。
