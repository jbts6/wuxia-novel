# 郭靖救师出手

## 描述
郭靖听到大师父柯镇恶在洞中应声，纵身而出，人随声至手起掌落抓住侯通海后心甩出。沙通天与彭连虎并肩攻上，梁子翁绕到身后欲施偷袭。
