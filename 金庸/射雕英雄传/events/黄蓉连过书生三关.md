# 黄蓉连过书生三关

## 描述
书生盘坐石梁尽头设三关考验。第一关：黄蓉巧解论语中冠者五六人童子六七人为五六得三十、六七四十二合为七十二贤。第二关：书生出字谜诗，黄蓉即猜出'辛未状元'四字。第三关：书生出上联'风摆棕榈千手佛摇折叠扇'，黄蓉对'霜凋荷叶独脚鬼戴逍遥巾'；书生又出绝对'琴瑟琵琶八大王一般头面'，黄蓉对'魑魅魍魉四小鬼各自肚肠'，书生大惊拜服让道。
