# 黄蓉在大殿阻拦瑛姑

## 描述
黄蓉在佛殿等候瑛姑，以老顽童消息和三道算题吸引其注意，用打狗棒法封字诀阻路，再布油灯竹签阵考较记心。瑛姑换签时暗中捏断竹签，随后大步踏断闯入后院
