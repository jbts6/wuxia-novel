# 瀑布洞口激战守护武穆遗书

## 描述
完颜洪烈率众入宫，沙通天率先钻入瀑布发现铁门。郭靖黄蓉抢先潜入洞口守卫：黄蓉以打狗棒法击败梁子翁，郭靖以掌力逼退沙通天，侯通海强冲被打出，灵智上人冲入激斗后袈裟铜钹均飞出。欧阳锋以蛤蟆功破铁门，与郭靖在洞口以空明拳拆招印证。完颜洪烈用激将法令欧阳锋全力出击。
