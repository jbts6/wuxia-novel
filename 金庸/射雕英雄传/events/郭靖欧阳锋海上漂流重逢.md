# 郭靖欧阳锋海上漂流重逢

## 描述
大船沉没后二人被潜流带出数里，在海中各被抓住后放开，同抱断桅漂流数日。遇海鱼以掌力击晕分食生鱼渡日，同桅共济。海流将二人送至同一荒岛，正遇上巨岩坠落压住欧阳克
