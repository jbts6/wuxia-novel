# 哲别与博尔术比箭

## 描述
博尔术请命与哲别比箭。哲别以有弓无箭相让，博尔术骑铁木真白口宝马持大汗弓箭出战。两人马上对射精彩纷呈，哲别接箭还射、镫里藏身、马腹射箭，射杀白口名驹。最终博尔术以无头箭射中哲别后颈，哲别亦以无头箭射中博尔术后心，双方惺惺相惜。哲别归降铁木真，以无头箭一命换郭靖平安。
