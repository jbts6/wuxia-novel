# 铁木真许婚亲上加亲

## 描述
铁木真为维系两家和气，主动将幼女华筝许配都史，王罕又提议将大孙女给术赤，亲上加亲。桑昆心中不乐意但不敢违父命。华筝为此好生不乐，每想到要嫁给都史便不高兴。
