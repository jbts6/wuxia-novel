# 王处一救郭靖震慑群雄

## 描述
王处一飞身而出以拂尘卷住彭连虎手腕救下郭靖，左足踏地深陷近尺震慑全场，识出完颜康为丘处机之徒后厉声质问，完颜康惶急邀其赴府
