# 一灯讲述往事

## 描述
一灯大师向众人讲述与刘贵妃、周伯通的往事：周伯通在园中教刘贵妃点穴功夫，二人日久生情，周伯通拒绝娶刘贵妃后出走，刘贵妃生下周伯通之子，孩子被蒙面刺客打伤，一灯因嫉妒拒绝救治，刘贵妃亲手杀死孩子后离去，一灯大彻大悟出家为僧
