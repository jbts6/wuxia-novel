# 鲁有脚率丐帮助郭靖西征

## 描述
鲁有脚率丐帮帮众千余人前来相助郭靖西征。郭靖苦恼不通兵法，鲁有脚提醒他身边有武穆遗书。此后郭靖挑灯夜读武穆遗书，遇不明处便请教鲁有脚，鲁有脚每出帐片刻便能解答疑难。
