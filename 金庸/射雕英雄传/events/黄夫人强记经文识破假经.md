# 黄夫人强记经文识破假经

## 描述
黄夫人看经书两遍即一字不漏记住，归还时指出经书是假的——已被欧阳锋掉包为占卜杂书。黄夫人当面背诵从小熟读的占卜之书与之对照，一字不错。周伯通大怒撕毁假经。
