# 漂流至荒岛与洪七公传位

## 描述
小舢舨随海流漂至荒岛。洪七公告知黄蓉自己蛇毒加掌伤武功已废，将丐帮帮主之位传给黄蓉为第十九代帮主，交予绿竹棒，又口传心授三十六路打狗棒法。洪七公晕倒后欧阳克企图对黄蓉施暴，被洪七公拄棒现身余威震慑而逃
