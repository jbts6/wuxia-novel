# 郭靖铁牢救穆易，杨铁心认出义兄之子

## 描述
郭靖前往铁牢欲救穆氏父女，自报姓名后杨铁心认出义兄郭啸天之子，激动落泪；此时王妃包惜弱赶来释放穆氏父女
