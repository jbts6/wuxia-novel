# 第三道试题：背诵九阴真经

## 描述
黄药师取出亡妻冯蘅手抄的九阴真经下卷让二人背诵，黄蓉打岔令欧阳克记不住，郭靖则滚瓜烂熟完整背出，黄药师以为亡妻显灵
