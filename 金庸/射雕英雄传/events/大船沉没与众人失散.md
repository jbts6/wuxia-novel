# 大船沉没与众人失散

## 描述
黄蓉救洪七公入海脱离火海，攀住小艇逼欧阳克拉洪七公上艇。山崩般巨响后半截大船沉入海中形成大漩涡，郭靖与欧阳锋随船沉没无影无踪。黄蓉在海中搜寻十余圈无果，筋疲力尽后昏厥
