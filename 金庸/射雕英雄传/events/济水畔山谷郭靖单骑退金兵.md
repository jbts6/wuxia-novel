# 济水畔山谷郭靖单骑退金兵

## 描述
郭靖见金兵烧村掳掠大怒，纵马以左右互搏之术单骑守在山谷隘口，右手刺左手砍抵挡数十金兵。蒙古兵赶至合力歼敌，百夫长识出金刀驸马。拖雷率军追击金兵，与郭靖安答重逢。
