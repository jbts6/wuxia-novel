# 欧阳锋恩将仇报暗算洪七公

## 描述
洪七公救出欧阳锋后，后颈微微一麻，回头见蛇杖刚掠过，毒蛇满口鲜血。原来欧阳锋以蛇杖暗算恩人，毒蛇咬伤洪七公后颈。洪七公中蛇毒后又遭欧阳锋猛力一掌击在背心正中，口喷鲜血俯身跌倒，全身劲力尽失武功全废
