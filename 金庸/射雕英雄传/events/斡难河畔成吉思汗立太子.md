# 斡难河畔成吉思汗立太子

## 描述
成吉思汗召集诸王众将大会漠北，宣布立太子。术赤被问及意见时遭察合台辱骂为杂种，二人当场殴斗。经众将劝解，察合台推举窝阔台，术赤亦同意，拖雷无异言。成吉思汗确认立窝阔台为太子，大宴诸将庆祝。
