# 华筝丘处机到访与丘处机劝善

## 描述
成吉思汗宣召郭靖入殿，丘处机与华筝公主已在座。大汗以断戟击郭靖肩头泄愤后一笑置之。丘处机吟诗悲天悯人，耶律楚材照实译成蒙古语。丘处机以道德经'天道无亲常与善人''兵者不祥之器'等劝谏大汗少杀生，成吉思汗极不投机，令郭靖陪道长休息。
