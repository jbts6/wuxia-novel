# 皇宫养伤往事

## 描述
洪七公讲述在皇宫御厨连吃四次鸳鸯五珍脍等名菜，被侍卫发现后逃到萼绿华堂养伤十余天，周伯通在宫中到处搜寻，最终在半夜装鬼哭猫叫找到他。又讲周伯通抓着梁子翁等恶贼饱打一顿逼其寻找郭靖黄蓉，搓泥丸假毒药制服众人的趣事。
