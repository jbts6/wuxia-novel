# 瑛姑刺中假扮一灯的郭靖

## 描述
瑛姑闯入禅房，见蒲团上坐著银须老僧（郭靖假扮），蒲团前放有肚兜和玉环。她心中刚硬提起匕首刺入胸口直没至柄，郭靖以两指捏住刃锋仅入肉半寸。瑛姑转身欲逃时见真正一灯立于门口
