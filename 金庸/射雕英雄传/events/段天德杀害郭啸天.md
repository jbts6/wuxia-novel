# 段天德杀害郭啸天

## 描述
郭啸天为保妻子投降被缚，段天德鞭打杨铁心。郭啸天飞脚踢段天德救义弟，段天德下令乱枪戳死，用锯齿刀砍断郭啸天右臂。郭啸天伤重气绝，临终嘱杨铁心去救弟妇与嫂子保全郭家骨血
