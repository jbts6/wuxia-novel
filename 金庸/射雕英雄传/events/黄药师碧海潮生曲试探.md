# 黄药师碧海潮生曲试探

## 描述
箫声调子忽变柔靡万端后急促催舞，郭靖以内功心法抵御后不受影响。暗中发现一长发长须老人（周伯通）盘膝运功抵御箫声，气喘愈急痛苦难当。郭靖抢上按住其大椎穴助其镇定，黄药师在暗中怒骂'小畜生坏我大事'箫声突止。郭靖运用马钰所授内功心法，物我两忘直到天明。
