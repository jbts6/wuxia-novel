# 周伯通被毒蛇咬伤

## 描述
周伯通夜间练拳时双手分打两套拳法分心，被突然出现的青蝮蛇咬中小腿。蛇毒猛烈，周伯通小腿肿粗倍余，全身酸软晕厥。岛上十五年从未见过毒蛇，疑为欧阳锋所驱。
