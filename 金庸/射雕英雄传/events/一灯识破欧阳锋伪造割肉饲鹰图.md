# 一灯识破欧阳锋伪造割肉饲鹰图

## 描述
一灯审视画作后拿到阳光下透视纸质，又取天竺僧所赠大庄严论经比对，发现画纸与经书同为西域纸张。一灯断定此画为西毒欧阳锋伪造，处心积虑以九阴真经设下毒计。黄蓉亦从笔法墨色分析出画非瑛姑所绘。
