# 周伯通赴桃花岛问罪被囚十五年

## 描述
周伯通带经书赴桃花岛责问黄药师，得知黄夫人已逝。因言语不当激怒黄药师被打伤囚于洞中。黄药师打断其双腿，逼其交出上卷经书。周伯通守在洞口，二人对耗一十五年。
