# 周伯通讲述华山论剑往事

## 描述
周伯通讲述五绝在华山绝顶论剑七天七夜，王重阳夺得天下第一与九阴真经。王重阳将经书压于蒲团下石板下不练其中功夫，武林因此安静一时。王重阳临终安排教中大事，嘱门下不可习练经中武功，欲焚毁经书但终不忍。
