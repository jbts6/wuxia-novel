# 郭靖以承诺求赦全城百姓

## 描述
成吉思汗下令屠城，十余万百姓惨遭杀戮。郭靖胸中热血沸腾，以大汗先前'不论你求甚麽都允可'的承诺，求他饶恕数十万百姓性命。成吉思汗大怒手按刀柄，郭靖身子发抖但仍坚持'我不后悔'。大汗被其倔强折服，拔刀叫'收兵'，掷刀驰马回城，诸将对郭靖横目而视。
