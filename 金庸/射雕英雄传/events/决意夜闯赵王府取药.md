# 决意夜闯赵王府取药

## 描述
黄蓉指出城中药铺之药尽被赵王府搜去，附近市镇也未必有药，提议直接夜闯赵王府取药救治王处一。郭靖虽惧王府高手如云，但在黄蓉誓言同生共死的鼓励下毅然同意，二人携手回城向王府进发。
