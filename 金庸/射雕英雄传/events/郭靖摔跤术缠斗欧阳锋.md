# 郭靖摔跤术缠斗欧阳锋

## 描述
郭靖不顾自身安危纵身抱住欧阳锋头颈，胁下被反手扫中仍不放手，以蒙古摔跤术近身缠斗：先使骆驼扳扳住其后颈，又以断山绞双手互叉绞住后颈令其陷于绝地，欧阳锋以上乘轻功从郭靖胯下翻出才得脱困。二人滚倒甲板，衣发满是火焰
