# 二十四桥明月夜

## 描述
黄蓉做蒸豆腐'二十四桥明月夜'，用兰花拂穴手功夫将嫩豆腐削成二十四个小球放入火腿圆孔中蒸制，火腿鲜味全入豆腐后弃火腿不食。洪七公大为倾倒。
