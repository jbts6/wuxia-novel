# 朱聪发现匕首与杨康之名

## 描述
朱聪以空手入白刃抢走郭靖匕首，发现刃身蓝光特异、柄上刻有'杨康'二字。全金发推测与杨铁心有关，朱聪归还匕首后追问，但郭靖不知杨康是谁。
