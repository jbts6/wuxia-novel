# 黄蓉在墓室得知花船之计

## 描述
黄蓉独处母亲墓室中，半夜听到父亲到墓前喃喃述说亡妻。黄药师透露花船入海必沉的真相：船底以生胶绳索胶缠而非铁钉，他本欲驾此船与妻子一齐葬身万丈洪涛。郭靖三人强要坐此船出海，等于烧了两部活的真经偿了亡妻心愿。
