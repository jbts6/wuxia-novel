# 黄蓉舍命救护柯镇恶

## 描述
柯镇恶腿中箭倒地，军官举刀砍来，黄蓉奔近挡在前面以竹棒护住头脸，羽箭射在她软猬甲上。她以兰花拂穴手拂中柯镇恶肩贞穴制住他以便救护。后截两名官军做成抬床抬柯镇恶，行出三十余里到农家休息，为他换金创药，又到铁枪庙投宿，打掉铁枪枪头给他当铁杖，赠田七鲨胆散。
