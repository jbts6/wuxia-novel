# 靖蓉秃木峰顶重逢

## 描述
郭靖登至秃木峰顶，万年寒冰结成琉璃世界。黄蓉自冰岩后现身，两人重逢相拥，滑倒在冰上。黄蓉身着当年张家口订交时郭靖所赠黑貂裘。二人在冰洞中诉说离情，黄蓉讲述从欧阳锋手中逃脱的经过（借归云庄五行八卦之阵脱身），冰释前嫌。
