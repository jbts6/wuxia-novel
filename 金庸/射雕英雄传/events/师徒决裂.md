# 师徒决裂

## 描述
柯镇恶将门牙和血吞入腹中，怒斥郭靖忘恩负义。郭靖欲砍断左腕谢罪被柯镇恶挡开，柯镇恶令其去割黄药师和黄蓉的头来见。黄蓉以打狗棒法'恶狗拦路'挡开柯镇恶再击的铁杖，柯镇恶悲愤向北疾驰而去。
