# 棒打参仙老怪

## 描述
梁子翁取出透骨钉欲射入洞中，被黄蓉石子击落手腕。梁子翁攻黄蓉欲夺竹棒，黄蓉以打狗棒法'棒打狗头'一招击败梁子翁，梁子翁见是打狗棒法惊恐逃走，招呼两名弟子转身便奔。
