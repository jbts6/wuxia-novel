# 四人夜入皇宫御厨

## 描述
二更时分四人翻入宫墙，洪七公指路至嘉明殿之东御厨房。洪七公坐梁上等候厨子做鸳鸯五珍脍，周伯通答应留下陪洪七公，让郭靖黄蓉去监视完颜洪烈。
