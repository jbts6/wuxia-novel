# 桃花岛恳谈惧祸

## 描述
黄蓉反复追问郭靖若六位师父、母亲、朋友都逼他杀她时怎么办，述说柯镇恶要杀她的恐惧。黄蓉伏在郭靖肩头啜泣，坦言不知害怕什么，只是想到柯镇恶的神情便心中慌乱，总觉得有一天郭靖会听师父的话杀了她。
