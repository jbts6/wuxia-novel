# 崖顶月夜遇梅超风

## 描述
郭靖与道人(马钰)在崖顶发现骷髅头后不久，梅超风从崖背攀上练功，全身关节格格作响，舞动四丈银色软鞭，鞭头十余只倒钩灵便如有手，马钰及时将郭靖扑倒救他一命
