# 雷雨黑暗中柯镇恶独战陈玄风

## 描述
雷雨骤至乌云掩月伸手不见五指，柯镇恶凭极灵耳音占尽优势，连中陈玄风两杖。陈玄风借闪电光亮欺身抓住铁杖，一拳击中柯镇恶胸口，掷铁杖如标枪飞射。韩宝驹金龙鞭卷住铁杖再救大哥。
