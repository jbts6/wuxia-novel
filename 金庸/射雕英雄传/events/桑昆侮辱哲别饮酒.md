# 桑昆侮辱哲别饮酒

## 描述
哲别进帐谢赐酒，桑昆当众阻止其用金杯饮酒，施以蒙古习俗中极大的侮辱。哲别又惊又怒，铁木真隐忍替哲别饮下酒水，化解尴尬。
