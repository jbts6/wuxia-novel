# 郭靖黄蓉入庙拜见一灯大师

## 描述
经竹林至石屋，见两僧人对坐蒲团。黄蓉猜出长眉僧人即段皇爷、现号一灯，率郭靖下拜称师伯。一灯喜见故人之后，问及黄药师与洪七公近况，又提及华山论剑往事。
