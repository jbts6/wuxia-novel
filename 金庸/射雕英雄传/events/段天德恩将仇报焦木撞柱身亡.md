# 段天德恩将仇报焦木撞柱身亡

## 描述
段天德趁众人受伤欲砍杀丘处机，焦木掷乌焦木打落其三颗牙齿，段天德恶性大发恩将仇报砍翻焦木和两名小沙弥。李萍扑上厮打未能阻止，焦木悔恨交加撞柱脑浆迸裂立时毙命，段天德拉李萍仓皇逃走
