# 君山丐帮大会争端

## 描述
丐帮四长老聚议争执帮主归属，净衣三老拥杨康，鲁有脚主张查明真相。彭长老率众欲擒黄蓉，郭靖跃上台护她。黄蓉朗声辩解洪七公尚在人世，杨康不知打狗棒真名露馅，群丐对其怒目相视。
