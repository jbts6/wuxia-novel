# 欧阳锋送人头讨好黄药师碰壁

## 描述
欧阳锋掷一包袱给黄药师，内有一书院儒生首级，自称杀了讲忠臣孝子的腐儒以讨好黄药师。黄药师变色称平生最敬忠臣孝子，挖坑埋人头恭敬作三个揖，凛然道'忠孝乃大节所在，并非礼法'
