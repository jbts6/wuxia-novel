# 穆念慈以死拒苟合

## 描述
完颜康情热如火欲解穆念慈衣带，穆念慈以五成真力格开双手，跃下地抢过断枪头对准胸膛垂泪道'你再逼我我就死在你面前'。声明自己虽是贫家女子但非不知自爱之人，若有洞房花烛之日自然如其所愿但今日若想轻贱于己有死而已。完颜康暗暗起敬认错，穆念慈约定牛家村等候迎娶。
