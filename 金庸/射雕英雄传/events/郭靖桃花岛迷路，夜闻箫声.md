# 郭靖桃花岛迷路，夜闻箫声

## 描述
郭靖随黄蓉登岛后在花丛中迷失方向，小红马也走失。独自在花树中迷路至天黑，饥饿疲倦沉沉睡去。中夜被箫声惊醒，循箫声前行，在白花丛中见到冯氏墓碑，恭恭敬敬跪拜四拜。
