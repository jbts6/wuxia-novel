# 白雕报信郭靖黄蓉骑马救人

## 描述
白雕飞来啄住郭靖衣服向外拉扯，郭靖神色惊惶称义兄义妹有难。黄蓉让小红马回去接黄药师，二人骑小红马由双雕领路赶到树林，发现拖雷、华筝、哲别、博尔术被绑在树上，欧阳锋与裘千仞在前，一名大宋将军已被欧阳锋掌力击毙。
