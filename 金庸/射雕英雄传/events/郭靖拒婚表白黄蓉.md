# 郭靖拒婚表白黄蓉

## 描述
郭靖拒绝娶华筝公主和穆念慈，众人追问其意中人。韩小莹猜出是穿白衫子的小姑娘黄蓉，柯镇恶和丘处机惊问是否黄药师之女。韩宝驹逼郭靖发誓永不见黄蓉，郭靖跪地流泪不肯屈从。韩小莹忆起亡故张阿生，心中感叹。
