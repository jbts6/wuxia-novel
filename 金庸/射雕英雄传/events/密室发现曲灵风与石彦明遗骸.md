# 密室发现曲灵风与石彦明遗骸

## 描述
黄蓉与周伯通进入密室，发现两具骸骨：一具仰天躺着肋骨齐断（曲灵风），一具伏在铁箱上被尖刀穿肋而死（石彦明）。通过铁八卦和刻'曲'字尖刀确认死者身份，黄蓉认出曲灵风为师哥。
