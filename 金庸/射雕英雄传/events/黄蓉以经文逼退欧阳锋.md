# 黄蓉以经文逼退欧阳锋

## 描述
黄蓉设计让欧阳锋取出假九阴真经经文，以焚烧经文相威胁，逼欧阳锋承诺放她走。欧阳锋虽答应但不肯真放黄蓉，因黄蓉背得出一灯大师所授译文，是唯一能为他解经之人。
