# 江南六怪赶到救援

## 描述
柯镇恶等六怪赶到，加入战团。六怪结成圆阵背里面外抵御，但沙、彭、欧阳三人武功太强，险象环生。韩宝驹肩头受伤仍咬牙坚持。郭靖以排云推月击彭连虎后心救韩宝驹。
