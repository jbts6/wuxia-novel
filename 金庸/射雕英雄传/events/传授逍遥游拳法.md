# 传授逍遥游拳法

## 描述
黄蓉要求学一套胜过郭靖的武功。洪七公教其'逍遥游'拳法，共三十六招，大袖飞舞东纵西跃身法轻灵之极。黄蓉不到两个时辰全数学会，两人并肩同使如玉燕大鹰翩翩飞舞。洪七公称此拳法郭靖不能学否则变成'苦恼爬'。
