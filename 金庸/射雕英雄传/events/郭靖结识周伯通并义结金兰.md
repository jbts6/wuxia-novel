# 郭靖结识周伯通并义结金兰

## 描述
天明后长发老人自报家门为周伯通，王重阳师弟，被困桃花岛十五年。二人交谈甚欢，周伯通试郭靖掌力后惊讶他如何能抵御碧海潮生曲。周伯通主动提议结拜为兄弟，郭靖推辞辈分不合但周伯通执意不拘礼法，二人向天誓盟结为兄弟。黄药师老仆送来酒菜，聋哑仆人的惨状令郭靖震惊。
