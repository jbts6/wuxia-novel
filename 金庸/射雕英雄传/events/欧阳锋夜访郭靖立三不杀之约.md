# 欧阳锋夜访郭靖立三不杀之约

## 描述
欧阳锋潜入郭靖军中寻找黄蓉，与郭靖交手后立下三不杀之约：郭靖落其手三次饶其不死，欧阳锋不得伤黄蓉毫发。鲁简梁三长老赶到合力对抗，欧阳锋见不易取胜遂约定后离去，声称十日内携蛇杖再来。
