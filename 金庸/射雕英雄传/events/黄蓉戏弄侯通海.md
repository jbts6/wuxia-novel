# 黄蓉戏弄侯通海

## 描述
侯通海数次抓黄蓉头发均落空，黄蓉将蛾眉钢刺插入头髻诱其上当，侯通海一把抓去手掌被钢刺对穿，痛得大跳；黄蓉笑称'你头上三只角斗不过我头上一只角'。
