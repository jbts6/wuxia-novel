# 颜烈欺骗包惜弱北上

## 描述
颜烈谎称杨铁心已被官兵杀害，以报仇为名取得包惜弱信任。包惜弱性格柔弱缺乏主见，孤苦无依之下只得随颜烈北上。二人经长安镇、硖石镇到达嘉兴秀水客栈，颜烈处处殷勤伪装至诚君子
