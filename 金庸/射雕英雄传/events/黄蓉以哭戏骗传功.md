# 黄蓉以哭戏骗传功

## 描述
洪七公扬长而去后在树上偷听。黄蓉假哭诉说无人保护、对头厉害，提及洪七公有一套武功天下无双古今独步连王重阳也忌惮三分，故意记不起名字。洪七公在树顶忍不住大喝'叫做降龙十八掌'跃下，黄蓉暗自得意。
