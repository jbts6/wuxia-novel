# 王处一识破穆念慈武功出自洪七公

## 描述
王处一见穆念慈起身迈步，忽地一按一扳试其武功。穆念慈受力后不向后跌反而向前俯跌，正合洪七公'刚猛绝伦、遇强愈强'的武功要旨。王处一识出她曾得九指神丐指点三日，叹道'这三天抵得旁人教你十年二十年'。
