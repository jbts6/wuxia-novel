# 郭靖割臂放血救周伯通

## 描述
郭靖先为周伯通吮吸毒液，后灵机一动想到自己血中或有克制蛇毒之物（因曾喝大蝮蛇血），拔匕首割臂放血于青瓷大碗中，撬开周伯通牙齿灌入血水，自己放血过多酸软昏睡。周伯通终得救。
