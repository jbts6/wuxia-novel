# 郭靖力抗洪七公近三百招

## 描述
洪七公使铁箫给郭靖、自己空手对战，降龙十八掌一至九变共一百六十二掌，郭靖以铁箫作剑抵御，失箫后以空明拳配合降龙拳左右互搏，最终双掌相抵亢龙有悔对碰
