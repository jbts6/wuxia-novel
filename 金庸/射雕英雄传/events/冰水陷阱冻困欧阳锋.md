# 冰水陷阱冻困欧阳锋

## 描述
简长老再设陷阱，改用冰水浇淋之法。欧阳锋再入陷阱，鲁有脚（听身后黄蓉低喝'倒水'）下令倾倒雪水。天时酷寒，雪水立即结冰，欧阳锋被困于四五丈长七尺圆径的大冰柱中。郭靖饶其第二次，一个时辰后打碎冰柱放人。欧阳锋运功呕出三口黑血后恨恨而去。
