# 尹志平夜访试武投书

## 描述
全真教弟子尹志平奉丘处机之命北上投书，少年好事先行与郭靖交手试其武功。郭靖以分筋错骨手应对十余招，终被尹志平诱敌摔倒。柯镇恶怒其摔郭靖，将其抓住重重摔在地下。
