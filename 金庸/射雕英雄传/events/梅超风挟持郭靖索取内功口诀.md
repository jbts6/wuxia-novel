# 梅超风挟持郭靖索取内功口诀

## 描述
梅超风在被四人围攻时挟持郭靖坐在其肩头，向他询问九阴真经内功口诀。郭靖口述攒簇五行、和合四象、五气朝元、三花聚顶等要诀，梅超风苦思十余年不解的秘奥一旦得解而大喜，但因此分心被欧阳克和沙通天击中左肩右胁。
