# 包惜弱雪夜救颜烈

## 描述
包惜弱在后院发现雪地血迹追踪至屋后古坟，发现一名受伤的黑衣人（颜烈）肩中狼牙利箭奄奄一息。她心慈不忍，用小刀割开肌肉拔出箭头，以止血散救治，用门板拖回柴房，喂以鸡汤
