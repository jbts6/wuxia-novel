# 黄药师认曲灵风遗物收傻姑为徒

## 描述
黄药师见密室格局是自己独创，找到曲灵风血书遗禀，知其为讨好师父冒险入宫盗宝终被侍卫围攻而亡。他追问傻姑是否会打拳，傻姑展示偷学的六七招碧波掌法。黄蓉踢倒傻姑试其下盘，证实曲灵风未正式传授。黄药师令傻姑称黄蓉为姑姑，正式收归门下，决意亲自教她武艺诗琴奇门五行。
