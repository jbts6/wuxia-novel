# 丘处机来信定比武之约

## 描述
朱聪朗读丘处机来信，信中赞七侠云天高义，告知杨康已于九年前找到并入门两年，约定两年后醉仙楼置酒比武。六怪深感己方处于下风，南希仁言'打不过也要打'以振士气。
