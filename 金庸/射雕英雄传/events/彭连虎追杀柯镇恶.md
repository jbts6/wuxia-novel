# 彭连虎追杀柯镇恶

## 描述
彭连虎趁郭靖被缠，纵到柯镇恶面前出言奚落。柯镇恶挥手发出铁菱，彭连虎惊弓之鸟般闪避。柯镇恶铁杖已失、足有残疾，勉力闪避，终被彭连虎双笔打中背心。
