# 偷听桑昆密谋

## 描述
马钰带郭靖跟踪梅超风至桑昆营帐，偷听到桑昆、札木合与完颜洪烈密谋暗害铁木真，完颜洪烈以封赏利诱二人，桑昆贪图南朝花花世界满口答应
