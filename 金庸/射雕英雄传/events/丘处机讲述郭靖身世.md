# 丘处机讲述郭靖身世

## 描述
丘处机从头讲述十八年前牛家村与郭杨二人结识、杀兵退敌、追寻遗孤、立约比武等往事。郭靖此时方知自己身世，伏地大哭。丘处机又说曾暗中传完颜康武艺数年，但其贪恋富贵油腔滑调。
