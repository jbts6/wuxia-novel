# 黄蓉惩戒恶夫妇

## 描述
黄蓉遇到骑驴的胖财主夫妇仗势欺人，以峨嵋钢刺割下胖妇人左耳，逼迫平时坐轿的胖夫妇反过来抬轿，让轿夫和丫鬟坐轿。黄蓉自号'姑娘大王'，扬长而去。
