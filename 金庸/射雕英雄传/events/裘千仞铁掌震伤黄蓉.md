# 裘千仞铁掌震伤黄蓉

## 描述
真正的裘千仞现身，黄蓉误认其为裘千丈不予躲避，裘千仞双掌猛拍黄蓉双肩将其震伤至气息闭绝。郭靖与裘千仞掌力硬拚落于下风，以神龙摆尾勉强挡住追击，抱黄蓉向峰顶飞跑。
