# 发现翠寒堂瀑布地形

## 描述
二人出御厨后循花香至翠寒堂，见银瀑从山边泻下注入池塘，堂前茉莉素馨等香花环绕。二人在凉床上嬉笑扮演皇帝，被护卫发现后躲入假山。
