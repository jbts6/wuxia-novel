# 白雕回报桃花岛有变

## 描述
双雕从桃花岛飞回，雌雕左胸中箭受伤，雄雕脚缚青布条但无金娃娃。黄蓉认出青布从父亲衫上撕下，推测桃花岛来了强敌，二人挂念黄药师安危决定告别下山
