# 杨铁心包惜弱双双殉义

## 描述
杨铁心夫妻被追兵围困，丘处机以一敌三力战不支。杨铁心不愿被擒受辱，回枪刺心自尽殉义。包惜弱扑向枪尖，以匕首刺胸而死，临终交出刻有'郭靖'二字的匕首。丘处机连称可惜，穆念慈掩面轻泣。
