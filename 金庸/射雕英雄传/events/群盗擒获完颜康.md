# 群盗擒获完颜康

## 描述
群盗掉头追击金国钦使船队，完颜康骁勇连伤数位寨主（贺、彭、董寨主受伤，缥缈峰郭头领被枪搠死）。水鬼凿穿船底，完颜康落水被擒。
