# 洪七公火中救欧阳锋

## 描述
着火大帆从天落下罩住欧阳锋，欧阳锋因灵蛇拳法被破茫然失神不及闪避，蛇杖被桅杆压住。洪七公不忍见其活活烧死，以铁锚锚爪钩住横桁将帆拉开救出欧阳锋，又抢住通红铁链防止扫到欧阳锋，双手被烫得嗤嗤作响
