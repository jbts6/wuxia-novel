# 裘千仞拜山震慑丐帮

## 描述
裘千仞率铁掌帮黑衣帮众拜山君山，与杨康握手时以铁掌灼热功捏晕杨康，夺简长老钢杖拗成弧形插入山石，展示铁掌功夫震慑全场。
