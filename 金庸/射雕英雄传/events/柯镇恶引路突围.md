# 柯镇恶引路突围

## 描述
柯镇恶要黄蓉给他竹棒，凭本地人经验辨蛇嘶箭声引路。发现西首小路近年已成竹林无法通行，丘处机王处一双剑开路斩竹。大路后大雨倾盆冲散浓雾，柯镇恶将竹棒交还黄蓉迳向东行。
