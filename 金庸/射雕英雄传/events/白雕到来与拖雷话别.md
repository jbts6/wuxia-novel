# 白雕到来与拖雷话别

## 描述
白雕从空中飞来认出主人，黄蓉欲摸反被啄。拖雷说明奉父命南下联合宋朝夹攻金国，华筝托他带雕给郭靖并捎话要郭靖尽早回蒙古。郭靖闻华筝之名心中茫然但不敢多想。拖雷因有将令不能延搁，与郭靖拥抱作别上马离去。
