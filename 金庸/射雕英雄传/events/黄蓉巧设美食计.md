# 黄蓉巧设美食计

## 描述
三人来到姜庙镇投店，黄蓉精心烹制'玉笛谁家听落梅'和'好逑汤'两道绝品菜肴。洪七公品尝后惊为天人，辨出五种肉味和樱桃中嵌的斑鸠肉。黄蓉以《诗经》'关关雎鸠在河之洲窈窕淑女君子好逑'解释汤名。洪七公大赞滋味远超皇宫御厨。
