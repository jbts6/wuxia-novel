# 段天德挟持李萍北逃

## 描述
段天德挟持李萍乘小船沿运河北逃，经扬州时被韩氏兄妹险些发现，又在山东利国驿差点暴露，李萍故意装疯留下形迹引追兵，段天德多次欲杀李萍未果。两人相打相骂一路北上至中都燕京。
