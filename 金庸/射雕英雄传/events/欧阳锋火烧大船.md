# 欧阳锋火烧大船

## 描述
欧阳锋在舱底引火，大船后梢火头冒起，船上男女乱窜帆飞樯舞。欧阳克抱黄蓉拉绳索溜入小艇，大船龙骨烧断折为两截沉入海中
