# 郭靖逼鲁有脚筹策与羊梯登峰

## 描述
郭靖以军法逼鲁有脚午时前筹划与黄蓉相见之策。鲁有脚指点秃木峰顶：今晚子时三刻黄帮主在峰顶相候。三长老率数十名亲兵以羊腿血冻在峰壁建'羊梯'，随杀随黏，直忙到半夜建至峰顶。
