# 火烧羊梯困欧阳锋于峰顶

## 描述
黄蓉佯装下峰，郭靖事先以漏油长索绕在羊腿之下。二人下峰后立即举火点燃长索，火蛇向上蜿蜒焚烧，羊腿受热血冰熔化，梯级自下而上逐一跌落。欧阳锋被困秃木峰绝顶。
