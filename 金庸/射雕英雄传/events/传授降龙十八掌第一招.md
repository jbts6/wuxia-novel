# 传授降龙十八掌第一招

## 描述
洪七公令郭靖跪下立誓不转授他人，郭靖以不愿对不起蓉儿为由婉拒学武。洪七公感动于其心地，决定只教一招'亢龙有悔'，演示一掌击断松树。详解精要：左腿微屈右臂内弯右掌划圈推出，掌法精要不在'亢'字而在'悔'字，有发必须有收，打出去十分力道留在自身还有二十分。郭靖苦练两个多时辰得大要。
