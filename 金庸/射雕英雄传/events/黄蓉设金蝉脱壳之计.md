# 黄蓉设金蝉脱壳之计

## 描述
黄蓉向四弟子提出冒奇险的救师之策：郭靖假扮一灯代受瑛姑一刀。郭靖点了一灯穴道，四弟子为郭靖剃光头、剃一灯胡子黏作白须，准备金蝉脱壳
