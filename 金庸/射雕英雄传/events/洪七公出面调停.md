# 洪七公出面调停

## 描述
洪七公从烟雨楼头轻飘飘落地，以威望震慑全场令众人住手。实则武功仅恢复三四成轻功，全无拳劲掌力。他以今日八月十四非比武正日为由令众人停战，明嘲暗劝全真诸子不要送死，命众人坐下练功等待明日比武。欧阳锋误以为洪七公武功恢复而心存忌惮，同意给面子
