# 三人跳上礁石脱险

## 描述
裘千仞跳水逃上岸，郭靖抓住要跳江追击的瑛姑，三人跃上江心礁石。黄蓉潜水取帆索、命雌雕绕柳树搭建绳桥，三人依次攀绳上岸
