# 唐诗剑法

## 描述
以唐诗诗句作剑招名字的剑法，万震山传授弟子

## 招式列表
| 招式 | 类型 | 描述 |
|------|------|------|
| [[孤鸿海上来，池潢不敢顾]] | 剑招 | 张九龄诗意化剑，飘逸自豪，顾盼间不屑旁视。戚长发误教为'哥翁喊上来，是横不敢过' |
| [[俯听闻惊风，连山若波涛]] | 剑招 | 闻风声而剑势如布匹般展开。戚长发误教为'忽听喷惊风，连山若布逃' |
| [[落日照大旗，马鸣风萧萧]] | 剑招 | 杜甫诗意化剑，壮阔萧瑟。戚长发误教为'老泥招大姐，马命风小小' |
| [[举头望明月，低头思故乡]] | 剑招 | 李白诗意化剑，拳招剑法皆可通用 |
| [[天花落不尽，处处鸟衔飞]] | 剑招 | 灵动飘逸的连刺剑招 |
| [[万国仰宗周，衣冠拜冕旒]] | 剑招 | 极具礼敬的起手剑招，长剑一立低头躬身。戚长发误教为'饭角让粽臭，一官拜马猴' |
| [[长安一片月，万户捣衣声]] | 剑招 | 秋夜意境化为剑招 |
| [[刺肩式]] | 剑招 | 后发先至，敌若出招进攻则破绽立露，刺中敌肩。老丐所授简化名 |
| [[耳光式]] | 剑招 | 剑交左手，右手反手打敌耳光，闪左打左闪右打右，越闪越重。老丐所授简化名 |
| [[去剑式]] | 剑招 | 以巧劲令敌长剑脱手飞出。老丐所授简化名 |
