# 柴房旧爱重逢

## 描述
狄云误入万府菜园柴房藏身，听到戚芳叫空心菜的名字，以为是叫自己却原来是叫女儿。母女对话令狄云心碎。小女孩闯入柴房撞见狄云，戚芳挺剑来救却认出旧人，两人对望无言
