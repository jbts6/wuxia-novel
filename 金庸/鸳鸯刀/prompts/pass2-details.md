# Pass 2 — 细节生成 prompt（《鸳鸯刀》专属）

## 角色

你是一位精通**金庸鸳鸯刀**的武侠小说研究者兼数据库工程师。你已经熟读《鸳鸯刀》全文，并且已经看过 Pass 1 生成的 5 类实体（characters / factions / locations / skills / techniques）。现在生成物品、对话和章摘要三类细节数据。

## 输入

1. Pass 1 的 5 个 JSON（characters / factions / locations / skills / techniques）——作为上下文锚点，保证 ID 一致。
2. `manifest.json`：章节清单（本书记为1章）。
3. 原文（已注入上下文）。

## 输出

3 个 JSON 文件：

- `items.json`：有情节意义的物品（兵器、秘籍、信物等）。
- `dialogues.json`：本章 5-15 条代表性台词。
- `chapter_summaries.json`：本章 150-250 字摘要。

## dialogues 选材指南

《鸳鸯刀》的经典对话场景（按情节顺序）：

1. **太岳四侠劫镖**（开篇段落）：
   - 盖一鸣自报家门："八步赶蟾、赛专诸、踏雪无痕、独脚水上飞、双刺盖七省"
   - 周威信的"江湖上有言道"系列俗语
   - 逍遥子给周威信改外号"铁鞭拜八方"

2. **袁冠南智取四侠银两**（中前段）：
   - 袁冠南以"阮囊羞涩"为由反向打劫太岳四侠
   - "书中自有黄金"的典故

3. **袁冠南斗卓天雄**（中段）：
   - 假称"五毒圣姑是我姑母"吓退卓天雄
   - "腐骨穿心膏"的虚张声势

4. **紫竹庵避难**（中后段）：
   - 林玉龙夫妇争吵不断，教授夫妻刀法
   - 袁萧二人在危急中互相回护

5. **萧半和寿宴**（后段）：
   - 袁冠南与袁夫人母子相认（翡翠狮子信物）
   - 萧中慧误以为袁冠南是亲兄的痛苦

6. **萧义身世揭秘**（结尾）：
   - "我萧义是个太监"——震撼全场的告白
   - 鸳鸯刀秘密揭晓："仁者无敌"

## 本书语言风格基线

- **叙述语言**：金庸标志性的半文半白，典雅流畅
- **周威信的对话**：大量引用"江湖上有言道"的俗语，体现其谨慎世故的江湖人性格
- **太岳四侠的对话**：滑稽夸张，盖一鸣的外号尤其冗长，常长风的墓碑梗，逍遥子的"料事如神"
- **袁冠南的对话**：书生气质，引经据典，偶尔故作高深
- **林任夫妇的对话**：吵闹不断，互不相让，但关键时刻又能配合
- **萧半和的对话**：前期豪迈大气，后期揭示身世时悲壮感人
- **卓天雄的对话**：傲慢自信，但被袁冠南假毒药吓退时暴露弱点

## chapter_summaries 重点

本书记为 1 章，摘要应覆盖以下主线和副线：

**主线：鸳鸯刀的争夺与归宿**
- 川陕总督命周威信护送鸳鸯刀进京
- 多方势力（太岳四侠、林任夫妇、萧中慧、卓天雄）争夺宝刀
- 最终太岳四侠擒获卓天雄，宝刀归还萧半和一方
- 鸳鸯刀秘密揭晓："仁者无敌"

**副线一：身世之谜**
- 袁冠南游学寻母，以翡翠狮子为信物
- 萧中慧的身世（杨伯冲之女，萧半和养女）
- 寿宴上母子相认，误以为兄妹

**副线二：萧义的太监身份与反清大志**
- 萧半和实为太监萧义，净身入宫刺杀皇帝
- 从天牢救出袁杨二夫人，乔装改扮十六年
- 最终身份暴露，官兵围剿，退守中条山

## items 焦点

《鸳鸯刀》的核心道具类型：

**关键剧情道具：**
- 鸳鸯刀（一短一长，藏有"仁者无敌"秘密）— 全书核心
- 翡翠狮子（袁冠南的寻母信物）— 推动身世揭秘
- 玉斑指（袁夫人给袁冠南的见面礼）— 触发母子相认
- 金叶子（袁冠南藏在书中的钱财）— 多次推动情节

**兵器（按使用者）：**
- 铁鞭（周威信，十六斤重）
- 峨嵋刺（盖一鸣）
- 流星锤（花剑影）
- 墓碑（常长风，顺手牵碑的临时兵器）
- 旱烟管（逍遥子）
- 弹弓（任飞燕）

**其他物品：**
- 沉香扇（萧半和给袁冠南的见面礼）
- 毛笔与墨盒（袁冠南的"武器"，实际用于虚张声势）
- 渔网（太岳四侠用来擒获卓天雄的工具）

## 硬性约束

### items

- **只收真正重要的物品**：推动剧情的关键道具、体现人物的标志性物品、武学相关的兵器等。
- 普通物品（路边买的刀、普通饭菜）不要凑数。宁缺毋滥。
- `description` ≥20字，必须基于原文描述，禁止模板。
- **必须为每个物品分配 `rarity_tier`**（凡品/良品/珍品/神品），不能全部设为"未知"。
  - 凡品：普通兵器、常见药物、日常工具（如墓碑、旱烟管）
  - 良品：名门正派的兵器、有特殊功效的药物（如铁鞭、弹弓）
  - 珍品：失传秘籍、罕见神兵、珍贵丹药（如鸳鸯刀——虽最终揭示秘密为"仁者无敌"，但其传说价值极高）
  - 神品：传说级神兵、失传绝学
- **source_refs 格式**：`{ chapter, anchor, event_type }`
  - `event_type` 选 `first_mention` / `climax` / `resolution` / `background`
- `owner` 可以是 Pass 1 的 `characters.json` 中的 char_id，也可以是 `factions.json` 中的 faction_id。
- `related_characters` 的 char_id 必须在 Pass 1 的 `characters.json` 里存在。
- `related_skills` 的 skill_id 必须在 Pass 1 的 skills.json 里存在。

### dialogues

- **强烈建议走"读原文提取"路径**：让 agent 直读 `ch_split/ch_001.txt`，挑 10-15 条代表性台词，保留原文一字不改。
- 如坚持用 LLM 凭记忆写台词（不推荐），必须严格遵守以下约束：
  - 挑 10-15 条最能体现角色性格、关系转折、情节张力的台词。
  - `text` 必须是原文一字不改的台词，不重写、不摘要。
  - `speaker` 的 char_id 必须在 characters.json 里存在。
  - `tone` 必须来自 `constants.md` 的 12 种 dialogue_tone，不能自创。
- 优先挑选：太岳四侠的经典台词、周威信的俗语、袁冠南的机智对白、萧义的身世告白、林任夫妇的吵嘴。
- 避免挑选：普通叙述性语句、无情节意义的过渡对话。

### chapter_summaries

- 本章 150-250 字，覆盖主要情节推进、人物变化、关系转折。
- 不要流水账，要抓主干（鸳鸯刀争夺 → 身世揭秘 → 萧义身份暴露 → "仁者无敌"）。
- `key_events` 列 3-5 条本章最重要的事件（每条 ≤20字）。
- `key_characters` 列本章出场的所有重要及以上角色的 char_id。

### 通用

- 禁止英文占位或问号兜底。
- 所有 ID 引用（char_id / skill_id / faction_id / loc_id）必须在 Pass 1 的 JSON 里存在；如不存在，先回查原文确认该实体是否该列入 Pass 1，或改用 null。

## 输出格式

每个文件是单个 JSON 数组，`JSON.stringify(data, null, 2)` 格式。用 `=== items.json ===` / `=== dialogues.json ===` / `=== chapter_summaries.json ===` 分隔。
