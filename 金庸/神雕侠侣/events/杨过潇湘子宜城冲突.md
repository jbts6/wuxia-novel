# 杨过潇湘子宜城冲突

## 描述
杨过与黄药师在宜城酒楼重逢，潇湘子与尹克西登楼，杨过掌掴潇湘子，黄药师以弹指神通击倒三人，与杨过切磋黯然销魂掌与弹指神通。
