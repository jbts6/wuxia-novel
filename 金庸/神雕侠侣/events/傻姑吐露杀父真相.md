# 傻姑吐露杀父真相

## 描述
杨过趁傻姑赤练神掌毒性发作神智迷糊之际，逼问杀父仇人。傻姑说出姑姑身上有毒针、姑姑叫郭靖靖哥哥，杨过由此认定郭靖黄蓉为杀父仇人，惊愤交迸
