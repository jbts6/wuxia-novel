# 杨过嘉兴王铁枪庙

## 描述
杨过至嘉兴王铁枪庙祭父，遇柯镇恶与沙通天等四人。柯镇恶讲述杨康往事真相，杨过得知父亲恶行后悲愤交加，救下柯镇恶并托其为父立碑。
