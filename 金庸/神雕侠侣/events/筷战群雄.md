# 筷战群雄

## 描述
金轮国师以筷子夹牛肉为引，依次与麻光佐、尼摩星、尹克西、潇湘子以内力较劲。震断麻光佐筷子，点穴退尼摩星，巧胜尹克西，力胜潇湘子，尽显蒙古第一国师之能
