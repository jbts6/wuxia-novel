# 小龙女以绸带金球接金轮国师攻势

## 描述
金轮国师要小龙女接其金轮十招以证武林盟主之能。小龙女以绸带金球施展古墓派绝顶轻功，在千钧一发之际以金球点穴、绸带卷柱借力等巧妙手段勉强接住攻势，杨过在旁虚报招数搅局
