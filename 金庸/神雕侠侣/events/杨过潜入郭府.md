# 杨过潜入郭府

## 描述
杨过携重剑潜入襄阳郭府，跟踪郭芙至其房外，后转至郭靖黄蓉居室窗外偷听，得知黄蓉生下双胞胎以及郭靖为断臂之事深感愧疚
