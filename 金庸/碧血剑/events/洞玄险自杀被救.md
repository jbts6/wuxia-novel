# 洞玄险自杀被救

## 描述
洞玄自忖必死，拔出匕首猛刺自己胸膛，被袁承志及时拦住夺下匕首。袁承志发现匕首与刺杀焦公礼的凶器一模一样，上刻仙都门下弟子字样，疑云大起
