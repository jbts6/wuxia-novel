# 袁承志准备入宫

## 描述
焦宛儿回报所见，袁承志决定持多尔衮密函潜入皇宫追查奸贼。焦宛儿扮书僮随行，罗立如也请求同往。吴平暗中劝罗立如成全师妹与袁承志的姻缘，罗立如虽心中怅惘但豁然释怀
