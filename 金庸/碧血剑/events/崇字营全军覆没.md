# 崇字营全军覆没

## 描述
清军阿济格、准塔率三旗精兵夜袭，崇字营寡不敌众全军覆没，孙仲寿、罗大千、朱安国、倪浩等首脑尽数阵亡。袁承志身中数箭昏迷被青青和何惕守寻回救治。
