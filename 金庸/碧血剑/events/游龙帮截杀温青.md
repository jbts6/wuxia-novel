# 游龙帮截杀温青

## 描述
温青携二千两黄金搭船，游龙帮沙老大率众截杀，温青剑法狠辣杀死沙老大及多人。荣彩率众追来，温青设计将金子丢入江中，实际以压舱石偷梁换柱
