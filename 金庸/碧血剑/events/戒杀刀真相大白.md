# 戒杀刀真相大白

## 描述
水云道人详述仙都派戒杀刀的来历：菊潭道长当年被各派车轮战围攻致死，此后门规规定满师弟子各赐一柄戒杀刀，训示'严戒擅杀，善视珍藏，义所不敌，举以自戕'。闵子华出示写于八月十八的信笺，证明戒杀刀早在焦公礼遇害前已被盗。焦宛儿恍然大悟，向闵子华赔礼道歉
