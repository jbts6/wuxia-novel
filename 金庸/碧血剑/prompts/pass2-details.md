# Pass 2 — 《碧血剑》细节生成 prompt

## 角色

你是一位资深的**武侠小说**研究者兼数据库工程师，尤其精通金庸、古龙、梁羽生、黄易四大家的作品。你尤其熟悉《碧血剑》——金庸早期代表作，以明末崇祯年间为历史背景。你已熟读本书，并且已经看过 Pass 1 生成的 5 类实体（characters / factions / locations / skills / techniques）。现在生成物品、对话和章摘要三类细节数据。

## 输入

1. Pass 1 的 5 个 JSON（characters / factions / locations / skills / techniques）——作为上下文锚点，保证 ID 一致。
2. `manifest.json`：章节清单。
3. 原文（已注入上下文）。

## 输出

3 个 JSON 文件：

- `items.json`：有情节意义的物品（兵器、丹药、秘籍、信物等）。
- `dialogues.json`：每章 5-15 条代表性台词。
- `chapter_summaries.json`：每章 150-250 字摘要。

## 书籍专属上下文

### dialogues 选材指南

《碧血剑》以下场景的对话最值得提取：

**经典对决场景**：
- 第7回：袁承志 vs 温氏五老，金蛇剑法初显威力
- 第8回：袁承志 vs 闵子华/洞玄（两仪剑法之战），袁承志冒充金蛇门人，狂态毕露
- 第10-12回：归辛树门人与袁承志的同门冲突
- 第20回：冯难敌等与何惕守的对峙

**情感转折场景**：
- 第5回：青青（温青青）向袁承志吐露身世，揭示金蛇郎君之女身份
- 第10回：阿九初登场，与袁承志的微妙互动
- 第17-18回：何红药讲述金蛇郎君往事，揭开夏雪宜与五毒教的恩怨
- 第20回：阿九断臂出家，袁承志与阿九的最终离别

**身世揭秘场景**：
- 第2-3回：崔秋山讲述袁崇焕旧事，袁承志得知身世
- 第11回：袁崇焕旧部重聚，追忆先帅
- 第17回：何红药讲述金蛇郎君的过去

**历史转折场景**：
- 第19回：闯军攻陷北京，崇祯自缢，明朝覆灭
- 第20回：李岩被逼死，闯军内部崩裂
- 第20回：袁承志率众远赴海外

### 本书语言风格基线

- **叙述者语言**：典雅白话，偶有文言句式，夹杂历史考据。如"其时正是崇祯六年，逆奄魏忠贤虽已伏诛"
- **袁承志说话**：谦逊有礼，自居晚辈，但关键时刻慷慨激昂。如"晚辈适才无礼，大是不该，谨向两位谢过"
- **青青说话**：伶俐刁蛮，爱争风吃醋，口齿犀利。如"有人要叫我亲爷爷啦！"
- **李自成说话**：豪迈粗犷，有草莽英雄气概
- **何惕守/何铁手说话**：娇媚中带毒辣，笑里藏刀。如"杀人虽然好玩，总得拣有武功的人来杀"
- **孙仲君说话**：暴躁跋扈，动辄出手
- **焦宛儿说话**：沉稳大方，处事得体
- **穆人清说话**：威严而慈祥，有宗师风范

### chapter_summaries 重点

**情节主线**：袁承志从懵懂少年成长为江湖群雄之首，为父报仇、行走天下，最终目睹明朝覆灭后携众归隐海外。

**情节副线**：
- 金蛇郎君往事线：通过金蛇秘笈、何红药回忆、温家庄恩怨逐步揭开
- 闯王起义线：李自成从起义到攻陷北京再到内部崩裂
- 阿九身份线：从江湖少女到揭明为崇祯之女长平公主

**摘要应突出**：
- 每回的主要情节推进
- 人物关系变化（尤其是袁承志与青青、阿九的感情线）
- 武功修炼与对决的关键进展
- 历史大事件与江湖故事的交汇

### items 焦点

《碧血剑》的核心道具类型：

1. **兵器类**：金蛇剑（金蛇郎君遗物，袁承志后使用）、铁鞭（冯氏兄弟使用）、青竹杖（程青竹/阿九使用）
2. **暗器类**：金蛇锥（金蛇郎君暗器）、含沙射影（五毒教机括暗器）
3. **秘籍类**：金蛇秘笈（记载金蛇剑法等武学的实体书册）
4. **信物类**：金丝镯子（安大娘赠袁承志）、玉簪（小慧的发饰）
5. **珍稀物品**：朱睛冰蟾（木桑道人所有，可解毒）

## 硬性约束

### items

- **只收真正重要的物品**：推动剧情的关键道具、体现人物的标志性物品、武学相关的秘籍/兵器等。
- 普通物品（路边买的刀、普通饭菜）不要凑数。宁缺毋滥。
- `description` ≥20字，必须基于原文描述，禁止模板。
- **必须为每个物品分配 `rarity_tier`**（凡品/良品/珍品/神品），不能全部设为"未知"。
  - 凡品：普通兵器、常见药物、日常工具
  - 良品：名门正派的兵器、有特殊功效的药物、珍贵的秘籍
  - 珍品：失传秘籍、罕见神兵、珍贵丹药
  - 神品：传说级神兵、失传绝学、极品丹药
- **source_refs 格式**：`{ chapter, anchor, event_type }`
  - `event_type` 选 `first_mention`（物品首次出现）/ `climax`（物品起关键作用的事件）/ `resolution`（物品的最终去向）/ `background`（背景提及）
- `owner` 可以是 Pass 1 的 `characters.json` 中的 char_id（人物拥有），也可以是 `factions.json` 中的 faction_id（门派拥有）。
- `related_characters` 的 char_id 必须在 Pass 1 的 `characters.json` 里存在。
- `related_skills` 的 skill_id 必须在 Pass 1 的 skills.json 里存在。

### dialogues

- **强烈建议走"读原文提取"路径**：让 agent 直接读 `<小说目录>/ch_split/ch_NNN.txt`，每章挑 5-10 条代表性台词，保留原文一字不改。
- 如坚持用 LLM 凭记忆写台词（不推荐），必须严格遵守以下约束：
  - 每章挑 5-15 条最能体现角色性格、关系转折、情节张力的台词。
  - `text` 必须是原文一字不改的台词，不重写、不摘要。
  - `speaker` 的 char_id 必须在 characters.json 里存在。
  - `tone` 必须来自 `constants.md` 的 12 种 dialogue_tone，不能自创。
- 优先挑选：人物初登场台词、关系转折对话、经典金句、冲突爆发点。
- 避免挑选：普通问路、吃饭、打招呼等无情节意义的对话。
- **《碧血剑》特别提醒**：本书第一回以张朝唐视角开篇，对话风格偏文人雅士；第2回起转为江湖风格。注意不同场景的对话基调差异。

### chapter_summaries

- 每章 150-250 字，覆盖主要情节推进、人物变化、关系转折。
- 不要流水账，要抓主干。
- `key_events` 列 3-5 条本章节最重要的事件（每条 ≤20字）。
- `key_characters` 列本章出场的所有重要及以上角色的 char_id。
- **《碧血剑》章节特点**：
  - 第1回：以张朝唐视角引出时代背景，是全书的"楔子"
  - 第2-4回：袁承志童年与拜师，节奏较慢
  - 第5-7回：温家庄事件与青青身世，节奏加快
  - 第8-12回：南京与山东江湖群雄汇聚，人物众多
  - 第13-18回：北京之行与五毒教冲突，主线推进
  - 第19-20回：闯军攻城与明朝覆灭，全书高潮与结局

### 通用

- 禁止英文占位或问号兜底。
- 所有 ID 引用（char_id / skill_id / faction_id / loc_id）必须在 Pass 1 的 JSON 里存在；如不存在，先回查原文确认该实体是否该列入 Pass 1，或改用 null。

## 输出格式

每个文件是单个 JSON 数组，`JSON.stringify(data, null, 2)` 格式。用 `=== items.json ===` / `=== dialogues.json ===` / `=== chapter_summaries.json ===` 分隔。
