# 白马啸西风 事件时间线

## 第 0 章

- **大漠追杀**: 吕梁三杰率六十三人从甘凉道一路追杀白马李三夫妇至回疆大漠，为夺取高昌迷宫地图...
- **白马李三力战殉难**: 白马李三重伤之下装死，突起反击手刃两名强敌，终因伤重不支力战而死...
- **上官虹殉夫**: 上官虹假意迎合史仲俊，趁其不备以金银双短剑刺入二人身体，一剑杀敌一剑自尽，当场气绝殉夫...
- **幼女逃入大漠**: 上官虹临终前将地图手帕塞入女儿怀中，挥鞭驱白马载着幼年李文秀飞驰逃命...
- **大漠风暴**: 一场遮天蔽地的大风暴席卷沙漠，李文秀下落不明，追兵损失人马，众人以为她已葬身风沙...
- **雪夜小屋遭遇**: 十二年后，陈达海风雪中闯入计老人小屋，与苏普、阿曼、李文秀等人相遇，威逼索要地图...
- **李文秀击败陈达海**: 李文秀施展华辉所传武功空手对战陈达海，以大擒拿手和叶底飞燕击败青蟒剑法，夺回阿曼...
- **成全苏普与阿曼**: 李文秀将阿曼从奴役中解救出来，主动成全苏普与阿曼，暗自伤心落泪...
- **雪地追踪**: 苏鲁克、车尔库率先锋队七人沿着雪地足迹追踪受伤逃走的陈达海，深入沙漠寻找高昌迷宫...
- **迷宫兜圈**: 七人在森林中追踪足迹，发现阿曼丢失的银镯出现在前方，意识到兜了圈子回到原路...
- **发现高昌迷宫**: 七人找到嵌在石山中的两扇铁门，阿曼转动门环打开迷宫大门，进入殿堂发现佛像与汉字碑文...
- **迷宫恶鬼杀人**: 瓦耳拉齐扮鬼在迷宫甬道中击毙骆驼和桑斯尔，七人惊恐逃出迷宫...
- **白衣鬼怪旷地显形**: 瓦耳拉齐披白袍扮厉鬼在旷地出现，隔空指杀一马一青年，震慑众人不敢逗留...
- **计老人之死**: 计老人赶来陪伴文秀，被瓦耳拉齐杀死，文秀悲痛不已...
- **瓦耳拉齐临终**: 瓦耳拉齐临终向文秀讲述篡改地图害盗伙的经过，欲以毒针留徒陪伴，因听到雅丽仙之名而放弃，最终死去...
- **哈卜拉姆解答婚嫁之问**: 苏鲁克问哈卜拉姆汉哈能否通婚，哈卜拉姆引用可兰经解答，认为只要善良便无妨碍...
- **李文秀骑白马东归**: 李文秀婉拒铁延部挽留，骑白马向东缓缓而行回中原，感叹心中所爱之人终究不能属于自己...
