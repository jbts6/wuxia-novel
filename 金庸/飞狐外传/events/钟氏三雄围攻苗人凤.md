# 钟氏三雄围攻苗人凤

## 描述
钟氏三兄弟着孝服到来，先以判官笔法围攻，苗人凤坐椅中以胡家刀法迎战，一刀云龙三现伤兆文、削断两枝判官笔；后三兄弟改用拳掌，兆英地堂拳断椅脚，苗人凤跃起掌击兆文兆能
