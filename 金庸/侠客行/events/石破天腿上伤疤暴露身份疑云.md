# 石破天腿上伤疤暴露身份疑云

## 描述
石破天捋起裤管展示左腿，果然有雪花六出形六点伤疤，他对自己身上伤疤全不知情极为震惊，连同肩头咬痕，身世之谜更加扑朔迷离
