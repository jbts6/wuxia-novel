# 陈冲之禀告擒获雪山派闯入者

## 描述
陈冲之禀报狮威堂擒获夜闯长乐帮的一男一女，疑为雪山派弟子，男子逃脱女子被擒。石破天误以为帮主身份，险些被陈冲之误解
