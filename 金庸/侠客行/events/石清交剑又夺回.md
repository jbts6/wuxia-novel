# 石清交剑又夺回

## 描述
石清将黑白双剑交予雪山派扣押，柯万钧欲强行夺剑反被内力震脱双腕，随后石清夫妇离去。雪山派七人携剑行走时遭官轿中人袭击，双剑被软鞭暗器夺回
