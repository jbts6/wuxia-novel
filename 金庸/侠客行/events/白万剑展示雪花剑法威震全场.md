# 白万剑展示雪花剑法威震全场

## 描述
白万剑拔剑在厅柱上瞬间连刺六剑形成雪花六出之形，展示雪山派剑法精妙，令全场肃然起敬，随后指出石中玉左腿上应有相同伤痕
