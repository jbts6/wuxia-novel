# Pass 2 — 细节生成 prompt（《天龙八部》专属）

## 角色

你是一位精通金庸《天龙八部》的研究者兼数据库工程师。你已经熟读本书，并且已经看过 Pass 1 生成的 5 类实体（characters / factions / locations / skills / techniques）。现在生成物品、对话和章摘要三类细节数据。

## 输入

1. Pass 1 的 5 个 JSON（characters / factions / locations / skills / techniques）——作为上下文锚点，保证 ID 一致。
2. `manifest.json`：章节清单。
3. 原文（已注入上下文）。

## 输出

3 个 JSON 文件：

- `items.json`：有情节意义的物品（兵器、丹药、秘籍、信物等）。
- `dialogues.json`：每章 5-15 条代表性台词。
- `chapter_summaries.json`：每章 150-250 字摘要。

## 硬性约束

### items

- **只收真正重要的物品**：推动剧情的关键道具、体现人物的标志性物品、武学相关的秘籍/兵器等。
- 普通物品不要凑数。宁缺毋滥。
- `description` ≥20字，必须基于原文描述，禁止模板。
- **必须为每个物品分配 `rarity_tier`**（凡品/良品/珍品/神品），不能全部设为"未知"。
- **source_refs 格式**：`{ chapter, anchor, event_type }`
- `owner` 可以是 Pass 1 的 `characters.json` 中的 char_id 或 `factions.json` 中的 faction_id。
- `related_characters` 的 char_id 必须在 Pass 1 的 `characters.json` 里存在。
- `related_skills` 的 skill_id 必须在 Pass 1 的 skills.json 里存在。

### dialogues

- **强烈建议走"读原文提取"路径**：让 agent 直接读 `<小说目录>/ch_split/ch_NNN.txt`，每章挑 5-10 条代表性台词，保留原文一字不改。
- 每章挑 5-15 条最能体现角色性格、关系转折、情节张力的台词。
- `text` 必须是原文一字不改的台词，不重写、不摘要。
- `speaker` 的 char_id 必须在 characters.json 里存在。
- `tone` 必须来自 `constants.md` 的 12 种 dialogue_tone，不能自创。
- 优先挑选：人物初登场台词、关系转折对话、经典金句、冲突爆发点。
- 避免挑选：普通问路、吃饭、打招呼等无情节意义的对话。

### chapter_summaries

- 每章 150-250 字，覆盖主要情节推进、人物变化、关系转折。
- 不要流水账，要抓主干。
- `key_events` 列 3-5 条本章节最重要的事件（每条 ≤20字）。
- `key_characters` 列本章出场的所有重要及以上角色的 char_id。

### 通用

- 禁止英文占位或问号兜底。
- 所有 ID 引用必须在 Pass 1 的 JSON 里存在。

## 《天龙八部》专属上下文

### chapter_summaries 重点

本书情节可分为三条主线和若干副线：

**段誉线**（1-18回为主，之后穿插）：
- 主线：大理王子离家出走 → 入江湖遇众女 → 发现身世秘密 → 最终为帝
- 重点突出：与木婉清、钟灵、王语嫣的感情纠葛；六脉神剑的修炼过程；段氏皇族内部纷争

**萧峰线**（14-28回为主，49-50回收束）：
- 主线：丐帮帮主 → 身世被揭 → 寻仇 → 误杀阿朱 → 辽国南院大王 → 拒绝征宋 → 自尽
- 重点突出：聚贤庄大战、阿朱之死、雁门关往事、辽国经历

**虚竹线**（29-38回为主）：
- 主线：少林小僧 → 珍珑棋局 → 得无崖子传功 → 缥缈峰收服群豪 → 西夏驸马
- 重点突出：天山童姥与李秋水的恩怨、梦姑之恋、灵鹫宫收服三十六洞七十二岛

**摘要应突出**：
- 三条线的交汇节点（如少室山大战三兄弟齐聚）
- 身世揭秘的关键章节
- 重大战斗场面
- 感情关系的转折点

### items 焦点

本书的核心道具类型：

**武功秘籍/经书类**（珍品/神品）：
- 北冥神功帛卷：段誉在无量山洞中发现，上有北冥神功和凌波微步图谱
- 易筋经梵文经书：少林寺至宝，游坦之意外获得
- 六脉神剑剑谱：大理天龙寺中以剑阵传承，无实体秘籍

**信物/关键道具**（良品/珍品）：
- 金钿小盒：钟灵母亲甘宝宝托段誉带给段正淳的信物
- 打狗棒：丐帮帮主信物，萧峰→吴长风→虚竹→新帮主
- 神木王鼎：阿紫用来驱使毒蛇的宝物，星宿派之物

**兵器类**（良品/珍品）：
- 修罗刀：秦红棉的兵刃，木婉清亦曾使用
- 鸠摩智的金轮：吐蕃国师的法器

**特殊物品**（珍品/神品）：
- 生死符：天山童姥控制下属的暗器（薄冰片），非实体物品而是武功产物
- 无崖子的珍珑棋局：以棋盘形式存在的机关
- 无量玉壁上的宝石剑：峭壁洞孔中悬挂的镶宝石长剑

## 输出格式

每个文件是单个 JSON 数组，`JSON.stringify(data, null, 2)` 格式。用 `=== items.json ===` / `=== dialogues.json ===` / `=== chapter_summaries.json ===` 分隔。
