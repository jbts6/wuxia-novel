# 萧峰误伤阿紫，以内力续命

## 描述
阿紫以毒针暗算萧峰在前，萧峰本能反应一掌将其震飞，阿紫经脉重伤气息奄奄。萧峰愧疚至极，连日以真气输入阿紫体内续命，数昼夜掌不离身
