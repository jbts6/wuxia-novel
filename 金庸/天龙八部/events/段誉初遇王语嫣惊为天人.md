# 段誉初遇王语嫣惊为天人

## 描述
段誉在曼陀山庄花林中听到王语嫣与阿朱对话，被其声音倾倒，莽撞闯出相见，见其容貌与无量山玉像一模一样，惊为天人，跪地称神仙姊姊
