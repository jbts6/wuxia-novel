# 神秘人以锁喉擒拿手击杀白世镜

## 描述
神秘人吹灭烛火后现身，以马大元家传锁喉擒拿手逼问白世镜，白世镜误以为是马大元鬼魂，吐出与马夫人合谋杀夫真相后被捏碎喉骨而死
