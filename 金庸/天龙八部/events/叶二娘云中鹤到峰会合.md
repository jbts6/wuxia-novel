# 叶二娘云中鹤到峰会合

## 描述
叶二娘抱着掳来的小儿幽幽上峰，面有血痕却颇具姿色。南海鳄神质问她排名。云中鹤随后到，被揭穿被两人围攻而非七大高手。三人在峰顶等待老大恶贯满盈，木婉清被点了昏睡穴后醒来苦等七日
