# 谭婆掌掴阿朱，谭公施药

## 描述
谭婆恼怒阿朱多嘴，欺身上前打了她一掌。谭公随即以极快手法取出金创药为她敷上，动作之快与谭婆打人一般无二，阿朱破涕为笑
