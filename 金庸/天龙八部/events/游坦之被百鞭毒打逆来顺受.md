# 游坦之被百鞭毒打逆来顺受

## 描述
阿紫因萧峰冷淡而迁怒游坦之，命室里鞭打百下。游坦之被打得人事不知，伤愈后被编入俘虏做粗活，却日夜盼望再见阿紫挨打也甘愿
