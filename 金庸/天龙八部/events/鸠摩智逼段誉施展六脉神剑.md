# 鸠摩智逼段誉施展六脉神剑

## 描述
鸠摩智解开段誉穴道逼他当众展示六脉神剑，段誉本不肯，但为救被火焰刀攻击的阿碧阿朱，终于出手，以中冲剑和少泽剑接下了鸠摩智的火焰刀
