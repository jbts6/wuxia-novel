# 虚竹被破门出教

## 描述
玄慈当众宣布虚竹犯杀淫荤酒四戒、私学旁门武功、擅任旁门掌门，玄寂宣布重打一百杖、废去少林武功、破门出教，虚竹伏地大哭愿代师父受三十杖
