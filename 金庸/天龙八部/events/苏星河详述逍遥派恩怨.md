# 苏星河详述逍遥派恩怨

## 描述
苏星河向虚竹详述逍遥派来历：师父有三位同门，师父排行第二因武功最强任掌门；苏丁二人门下，丁春秋因杂学不通与掌门无望而暗算师父，以看秘笈为名逼苏装聋作哑三十年
