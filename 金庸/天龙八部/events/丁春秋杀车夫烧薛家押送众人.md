# 丁春秋杀车夫烧薛家押送众人

## 描述
丁春秋命人雇了十辆驴车，将车夫全部杀死灭口，把玄难、邓百川、函谷八友等数十人塞入车中，放火烧了薛家，押送众人不知去向
