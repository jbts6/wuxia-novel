# 设计擒拿鳌拜

## 描述
康熙选三十名小太监练习摔跤，以考校为名召鳌拜入书房，茶盏摔落为号令十二名小太监一拥而上，韦小宝撒香灰迷眼、用香炉砸晕鳌拜，康熙亲手刺其背心，终将满洲第一勇士擒获
