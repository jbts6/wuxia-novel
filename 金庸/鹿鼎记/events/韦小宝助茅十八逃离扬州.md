# 韦小宝助茅十八逃离扬州

## 描述
韦小宝扶着受伤的茅十八走出丽春院，雇驴车前往得胜山。韦小宝拒绝茅十八的元宝，以'好朋友有福共享有难同当'回应，二人结为朋友。
