# 饭店怒斥吴三桂部下

## 描述
茅十八与韦小宝在饭店中遇到吴三桂手下十余名官员，茅十八当众痛骂吴三桂为大汉奸，双方大打出手。韦小宝钻桌底砍人脚板，白姓沐王府汉子暗中以竹筷相助击退敌手。
