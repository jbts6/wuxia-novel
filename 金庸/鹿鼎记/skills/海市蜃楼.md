# 海市蜃楼

## 描述
连发虚招的拳法，每一招既不打实又不用老，速度比平常拳法快数倍，意谓尽皆虚幻。双儿以此招迷惑罗刹勇士，使其连挡数下均挡空，再趁势点穴制敌。
