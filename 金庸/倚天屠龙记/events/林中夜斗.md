# 林中夜斗

## 描述
常遇春与张无忌在树林中休憩，夜半闻兵刃交击之声。少林、昆仑、峨嵋、海沙四派八人围攻明教彭莹玉，逼问白龟寿下落以追查谢逊踪迹。彭莹玉杀一少林僧，中蝎尾钩毒后装死重创五人。丁敏君刺瞎彭莹玉右眼，纪晓芙多番维护彭莹玉。丁敏君当众揭破纪晓芙隐秘，刺伤纪晓芙。白龟寿现身挡剑救彭莹玉，临死一掌击伤丁敏君。彭莹玉为保纪晓芙名声杀死所有知情者后离去。
