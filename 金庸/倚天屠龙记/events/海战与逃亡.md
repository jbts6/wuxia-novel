# 海战与逃亡

## 描述
波斯大船追来开炮。张无忌指挥还炮但落空。敌炮击中船头起火，又击碎船舵。梢公装填火药过多致大炮爆炸，梢公与水手炸死。众人改乘小船逃入大海。大船燃烧爆炸。
