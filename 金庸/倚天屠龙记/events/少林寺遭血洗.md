# 少林寺遭血洗

## 描述
明教上少室山拜山被拒，入寺后发现到处激斗痕迹和血渍，寺中无人。罗汉堂十八罗汉像背后刻"先诛少林，再灭武当，惟我明教，武林称王"十六字，有人将罗汉像转向墙壁暗助明教。
