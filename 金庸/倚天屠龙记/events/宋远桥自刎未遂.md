# 宋远桥自刎未遂

## 描述
宋远桥知悉宋青书弑师叔后悲痛欲绝，追杀宋青书未果，回剑自刎。张无忌施展挪移乾坤夺下长剑，剑尖仍在宋远桥项颈划出血痕。殷梨亭以金创药敷伤。
