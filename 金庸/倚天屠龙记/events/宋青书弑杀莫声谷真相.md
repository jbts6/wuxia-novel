# 宋青书弑杀莫声谷真相

## 描述
宋青书从丐帮逃走被追回，陈友谅与掌钵龙头当众揭露：宋青书深夜偷窥峨嵋诸女卧室，被莫声谷撞见追击，石冈比武中以震天铁掌打中莫声谷背部致其肋骨断裂身亡。武当四侠在岩后亲耳听到全部真相。
