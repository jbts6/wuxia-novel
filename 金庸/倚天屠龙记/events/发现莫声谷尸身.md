# 发现莫声谷尸身

## 描述
张无忌与赵敏藏身山洞内洞，张无忌碰到一具冰冷尸身，惊惶中抱着尸体走出，发现竟是七师叔莫声谷。莫声谷脸上全无血色、双目未闭，身上中的是内家掌力。
