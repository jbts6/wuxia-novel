# 赵敏坠崖

## 描述
赵敏骑马逃至悬崖，慌不择路连人带马坠入深谷。马当场震死，赵敏身未离鞍双手抱马颈仅昏迷。张无忌追至崖边攀下谷底，以九阳真气为她疗伤。
