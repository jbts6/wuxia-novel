# 赵敏调虎离山救张无忌

## 描述
赵敏见张无忌抱着莫声谷尸身处境尴尬，纵身舞剑闯出山洞，以峨嵋派拚命招数分刺武当四侠，夺马向东奔逃。实为调虎离山，使张无忌能藏好尸身脱身。
