# 倚天屠龙功

## 描述
张三丰情之所至，将'武林至尊，宝刀屠龙，号令天下，莫敢不从，倚天不出，谁与争锋'二十四字二百一十五笔演为一套拳法。每一字包含数招数般变化，笔划多不觉繁、少不见陋，凝重险劲、淋漓酣畅。张翠山从指书中领悟此功。
