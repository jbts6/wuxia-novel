# 七伤拳

## 描述
每拳含七股不同劲力：或刚猛，或阴柔，或刚中有柔，或柔中有刚，或横出，或直送，或内缩。对应体内阴阳二气、金木水火土五行，心属火、肺属金、肾属水、脾属土、肝属木。
