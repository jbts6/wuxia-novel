# 乾坤大挪移

## 描述
明教历代相传最厉害的武功，主旨颠倒一刚一柔一阴一阳的乾坤二气。先激发自身潜力，再牵引挪移敌劲，即「四两拨千斤」之极至。共七层，历代最高仅练到第五层即走火入魔，杨逍练至第二层，阳顶天练至第四层。
