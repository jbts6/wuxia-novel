# 卜沉沙天江夺走辟邪剑谱

## 描述
嵩山旁枝卜沉、沙天江潜入林家老宅佛堂，趁林平之岳灵珊寻找剑谱时出手点倒二人，从达摩画像所指屋顶取得辟邪剑谱袈裟
