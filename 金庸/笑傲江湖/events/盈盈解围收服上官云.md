# 盈盈解围收服上官云

## 描述
盈盈及时赶到，举黑木令假传教主旨意，放火扰乱贾布心神，令狐冲三人乘机冲入灵龟阁杀尽贾布手下，贾布被方证击落深谷身亡。盈盈以上官云升任青龙堂长老和三尸脑神丹收服上官云归降。
