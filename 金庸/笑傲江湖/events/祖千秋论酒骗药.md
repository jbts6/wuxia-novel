# 祖千秋论酒骗药

## 描述
落魄书生祖千秋登船，大论汾酒用玉杯、葡萄酒用夜光杯等酒具学问，将续命八丸混入八种美酒中骗令狐冲逐一饮下，桃根仙赌输吞下古藤杯
