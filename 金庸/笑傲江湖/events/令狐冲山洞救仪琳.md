# 令狐冲山洞救仪琳

## 描述
令狐冲在山洞外以笑声引开田伯光，潜入洞中救出仪琳，躲在草丛中肩头被砍一剑仍不吭声，后与田伯光在黑暗中搏斗多次受伤
