# 令狐冲擒仪琳之母吊于树上

## 描述
令狐冲横剑拦截仪琳之母，三剑迫退后桃谷六仙合力点穴擒住，剥树皮搓绳将其吊于高树，令狐冲在树上刻'天下第一醋坛子'
