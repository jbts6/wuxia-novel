# 令狐冲与黄钟公比剑

## 描述
令狐冲以玉箫代剑, 黄钟公以瑶琴代兵, 黄钟公施展七弦无形剑与六丁开山, 但因令狐冲内力全失琴音毫无效用, 最终七弦齐断大败
