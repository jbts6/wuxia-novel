# 任我行邀令狐冲入教遭拒

## 描述
任我行传令狐冲左臂通脉之法后重提入教之事，以吸星大法后患相胁。令狐冲斩钉截铁拒绝，指斥日月神教中尽是卑鄙龌龊之徒，并告知已答应定闲师太出任恒山派掌门。任我行大笑称其为小怪，盈盈随父不随冲。四人别过。
