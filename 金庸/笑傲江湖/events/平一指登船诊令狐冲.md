# 平一指登船诊令狐冲

## 描述
平一指受人之托登船为令狐冲搭脉，察觉其体内有八道异种真气（两道来自不戒和尚、六道来自桃谷六仙），宣布无法医治，赠十粒镇心理气丸延命百日
