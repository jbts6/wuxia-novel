# 田伯光透露被点穴下毒

## 描述
田伯光袒胸展示双乳下两枚死穴红点，告知被人点穴下毒，一个月内请不动令狐冲便毒发身亡，全身化为烂肉三年六个月方才烂死，唯平一指或下毒之人可解
