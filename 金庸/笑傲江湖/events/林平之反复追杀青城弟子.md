# 林平之反复追杀青城弟子

## 描述
林平之数次来袭，以快剑逐一刺杀于人豪、吉人通、方人智、贾人达等青城弟子，并纵马践踏贾人达。余沧海手抖茶壶盖响，内心恐惧万分。
