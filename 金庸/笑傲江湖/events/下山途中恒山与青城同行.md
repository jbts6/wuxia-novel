# 下山途中恒山与青城同行

## 描述
恒山派下山途中与青城派在小镇相遇。林平之夫妇骑马追来，林平之突施快剑刺死一名青城弟子后扬长而去，盈盈决定跟定青城派以观察林平之剑法。
