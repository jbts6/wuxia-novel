# 令狐冲受罚面壁一年

## 描述
岳不群追究令狐冲杀罗人杰、口出无礼之言、群玉院藏人等过错，罚其在玉女峰面壁一年。令狐冲对魔教中人嫉恶如仇之问无法回答。
