# 苗若兰赠玉马化解珠儿之争

## 描述
苗若兰以一对玉马赠予双童补偿损坏的宝珠，并巧思将碎珠嵌入玉马双眼。双童大喜但得知她是苗人凤之女后，将玉马退还，以示胡苗两家恩怨不可逾越。
