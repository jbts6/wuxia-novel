# 刀剑交换比武真相

## 描述
胡一刀主动提议交换刀剑比武，用苗家剑对苗人凤胡家刀，使胜负不涉两家武功威名。二人激战一天，互相指点招数不足，胡一刀凭聪明才智将苗家剑施展得如苦练数年。
