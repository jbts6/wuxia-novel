# 宝树揭示铁盒中的闯王军刀

## 描述
宝树从阮士中身上夺走铁盒，命曹云奇开启未果，田青文请周云阳以机括打开，盒中现出闯王军刀，鞘刻'杀一人如杀我父，淫一人如淫我母'军令，群豪震惊。
