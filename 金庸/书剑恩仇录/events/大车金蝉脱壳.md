# 大车金蝉脱壳

## 描述
赵半山钢镖毙敌无尘追魂剑砍杀镖师捕头，群雄抢到大车却发现车中是断臂捕头胡国栋而非文泰来。张召重用金蝉脱壳之计早已押走文泰来。
