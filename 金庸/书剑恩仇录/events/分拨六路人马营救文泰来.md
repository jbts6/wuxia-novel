# 分拨六路人马营救文泰来

## 描述
陈家洛分拨六路人马：余鱼同哨路联络，赵半山领章进石双英，无尘领杨成协蒋四根，陈家洛领卫春华心砚，陆菲青领孟健雄安健刚，周仲英领周绮徐天宏骆冰，分路进嘉峪关后会集。
