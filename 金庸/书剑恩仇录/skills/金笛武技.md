# 金笛武技

## 描述
余鱼同以金笛为兵器的武技，笛中暗藏短箭，可吹奏亦可攻击。

## 招式列表
| 招式 | 类型 | 描述 |
|------|------|------|
| [[金笛短箭]] | 暗器 | 金笛中暗藏短箭，胸气猛吐箭笔直疾飞，射瞎言伯干左眼 |
| [[疾风穿笛]] | 音攻 | 狂舞金笛时风穿笛孔发出呜呜声响 |
| [[同归于尽笛法]] | 搏命 | 全然不招不架尽攻要害，招招同归于尽笛笛两败俱伤 |
| [[金笛吹箭]] | 暗器 | 以金笛吹射短箭，可远程伤敌，射中面颊或后脑致命 |
| [[笛点穴道]] | 点穴 | 以金笛尖端点刺期门穴、气俞穴等穴道 |
