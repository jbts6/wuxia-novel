# 长亭对峙

## 描述
老人在长亭中以旱烟管暗藏玄机，上官金虹走近试探，两人以手指精微变化进行了一场无声的生死决斗。上官金虹三根手指的每一个变化都被老人封死，最终烟柱射向上官金虹面门后消散，上官金虹行礼退走。
