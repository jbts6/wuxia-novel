# 红鞋子小楼年度聚会

## 描述
阿土引陆小凤至荷塘小楼，红鞋子七位成员陆续到场。二娘展示年入一百八十万两的存折，三娘展示割下的七八十个鼻子。陆小凤在树上窥视，发现欧阳情竟是四姐，大为震惊。
