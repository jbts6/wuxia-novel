# 暗器突袭石秀雪身亡

## 描述
孙秀青正要说破青衣楼秘密时，窗外射入毒针击中其背，石秀雪扑向窗口亦被毒针射中倒地，西门吹雪抱起孙秀青穿窗追出，马秀真叶秀珠紧随其后
