# 天坛大殿四人伏击

## 描述
陆小凤与西门吹雪在天坛大殿调查时遭四人伏击：两个黄衣喇嘛从殿后现身，两个海南剑派黑衣刺客藏身木箱中偷袭。西门吹雪一剑刺瞎一人左眼后将其击杀，陆小凤以灵犀一指夹住偷袭之剑，箱中刺客被活活吓死，狞笑喇嘛被西门吹雪一剑穿喉，沉默喇嘛掷出九枚铜环后翻窗逃走
