# 豆腐店偶遇老实和尚

## 描述
破晓时分在小村豆腐店歇脚，陆小凤偶遇老实和尚。和尚透露认得公孙大娘但拒绝告知行踪，又警告二人三日内必被人打破脑袋，随后溜走。
