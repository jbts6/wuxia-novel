# 司空摘星现身立赌约

## 描述
伪装成伙计的司空摘星露面，受人之托要偷陆小凤的红缎子。陆小凤与他立下赌约：今夜司空摘星若偷走红缎子，陆小凤再替他挖六百八十条蚯蚓；若偷不走，司空摘星要叫陆小凤祖宗。
