# 丹凤公主揭穿逼迫之计

## 描述
丹凤公主直言揭穿山西雁约众人来是为了逼走陆小凤，让霍天青不战而胜。山西雁羞愤，问众人若陆小凤不走怎么办，小贩表示自己走。
