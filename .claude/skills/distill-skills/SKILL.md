# 提炼功法（旧版，仅供参考）

从 `<小说目录>/archive/skills.json` 提炼出 `<小说目录>/skills.json`，只保留真正的功法。

## 核心原则

**像编纂技能图鉴一样筛选。** 有独立名号、有传承体系、有武学价值的功法才配列入图鉴。一次性的动作、医术、训练法不配。

## 判断标准

**保留** — 满足以下任一条件：

| 条件 | 示例 |
|------|------|
| 有明确招式体系的武功 | 降龙十八掌、打狗棒法、蛤蟆功 |
| 门派标志性功法 | 玉女剑法、弹指神通、全真剑法 |
| 可传承的内功心法 | 玉女心经、九阴真经、寒玉床内功 |
| 有实战价值的独立招式 | 三无三不手、美女拳法、天罗地网势 |
| 阵法（有明确阵位和配合） | 天罡北斗阵、二十八宿大阵、九宫八卦棘藤阵 |
| 暗器功夫（有招式体系） | 金铃索法、双轮轮法、驭蜂术 |
| 精神攻击（属内功法门） | 移魂大法 |

**删除** — 满足以下任一条件：

| 类型 | 示例 |
|------|------|
| 医术/疗伤 | 接骨点穴、闻香点穴、杨过接骨、点穴止血 |
| 威慑/标记 | 血手印、赤练血手印 |
| 驯养/驱使 | 双雕驯令、玉蜂驯召、玉蜂指挥法 |
| 非武功特殊能力 | 小龙女琴音、指上刻石、慑心术、降魔伏妖咒 |
| 泛称/总类 | 全真武功、古墓派武功、桃花岛武功、丐帮武功、轻功身法 |
| 入门/基础 | 全真入门心法、全真入门拳法、全真入门套路、江南七怪根基功夫 |
| 训练法（非实战） | 古墓派绳上睡法、古墓派捉麻雀功 |
| 个人临场发挥 | 郭靖推劲、杨过飞石点穴、杨过装睡制穴、铁鞭汉子左掌 |
| 临时/一次性 | 杨过大剪刀破拂尘、杨过大剪刀破须 |
| 重复/已合并 | 李莫愁赤练神掌（已有赤练神掌）、黄药师玉箫剑法（已有玉箫剑法） |
| 太过细碎的单招 | 太公钓鱼、李莫愁拂尘巧劲、陆无双夺剑 |
| 伪装/易容 | 周伯通易容混战 |
| 情绪/音律攻击 | 李莫愁悲歌慑心、潇湘子鬼音 |

## 执行

1. 确认 `<小说目录>/skills.json` 存在且 `<小说目录>/archive/skills.json` 不存在
2. 将 `<小说目录>/skills.json` 移动到 `<小说目录>/archive/skills.json`
3. 读取 `<小说目录>/archive/skills.json`
4. 按上述标准逐条判断保留/删除
5. **交叉检查 techniques.json**：遍历待删除 skill，检查 `techniques.json` 中是否有 technique 指向该 skill。若 technique 名称合格（有独立招式名，非泛称/描述），则保留该 skill
6. 对保留条目应用命名规则
7. **游戏设计师过滤**：再过一遍保留清单——这个功法是否有资格列入技能图鉴？有独立名号、有传承体系、有武学价值的才留，凑数的删
8. 去重：去人名后同名功法合并为一条
9. 写入 `<小说目录>/skills.json`
10. 写入 `<小说目录>/skills_summary.md`

## skills_summary.md 格式

```markdown
# 《书名》功法清单

共 N 条（原 M 条，保留 X%）

| 功法 | 保留原因 |
|------|----------|
| 降龙十八掌 | 丐帮绝学，有明确招式体系 |
| 打狗棒法 | 丐帮绝学，有明确招式体系 |
| ... | ... |
```

保留原因从判断标准中选取最匹配的一条，用简短中文表述。

## 产物校验

- JSON 可解析
- 条目数约为 archive 的 30%–50%（视功法密度而定）
- 无重复 ID
- 所有 ID 为小写下划线
- `source_refs` 和 `rag_refs` 原样保留，不修改
- `skills_summary.md` 条目数与 `skills.json` 一致
